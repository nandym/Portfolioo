Angular Framework


The Website first started as a learning experience to grow my Front-End skills.
The learning experience was easy as I had tons of information and examples on the internet to approach this the way I wanted.
When first learning Angular on their website I was shown how handle the data from interfaces to create lists of items and choose which type of data was shown, this was very helpful, it meant I could easily add new Works to my portfolio by adding them to an Interface.









Interface used to pull data from where I query content from to create the boxes in the main page

The Design


When thinking of the design for the website I envisioned something very easy to traverse, with mostly images and little text. I wanted it to be sleek and clean, so my first image were the boxes from the first page.

Afterwards, the hard part was to translate my mental image into paper, while it might look good in my head it would look very different if I didn't flesh it out more.

In order to achieve something I liked I analysed shapes from other websites and made the first shapes, round shapes, slate, sharp shapes. After building on that I choose many colour pallets, but after a while I ended up creating my own and I cannot stress how important it is to have a colour pallet, it will inspire the theme of your website and flourish new ideas, this is applied to any project.












Mobile Usability



At the start mobile was already in mind, the design was already simple, so to adapt to Mobile devices changes were made to the CSS, a basic check if the user is in Portrait or Landscape mode allowed me to change it accordingly.
Mainly, the changes needed were to change paddings and make it so the grid of boxes only fit 1 in mobile and 2 when not on mobile, of course, also slight text adjustments
